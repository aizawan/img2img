# img2img
Tensorflow implementation for Image to Image Translation with Generative Adversarial Networks.

This repository is still in development.

## Environments
- Python 3.5.2
- Numpy==1.13.3
- Pillow==4.2.1
- tensorflow-gpu==1.3.0
- tensorflow-tensorboard==0.1.7
